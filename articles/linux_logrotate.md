* Administer, back up and monitor log files
* The logrotate program is used to provide the administrator with an up-to-date record of events taking place on the system
* The logrotate program is a log file manager. It is used to regularly cycle (or rotate) log files by removing the oldest ones from your system and creating new log files. It may be used to rotate based on the age of the file or the fileâ€™s size, and usually runs automatically through the cron utility.
* The logrotate program is configured by entering options in the /etc/logrotate.conf file.
